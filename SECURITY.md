# Security Policy

## Supported Version

Only the current `main` branch is supported. hoolah is a static browser app,
so fixes are shipped by rebuilding and redeploying the site.

## Reporting a Vulnerability

Please do not open a public issue for a vulnerability.

Use GitHub private vulnerability reporting if it is available on the repository.
If it is not available, contact the maintainer privately through the contact
address listed on https://angelicanaguio.com.

Helpful reports include:

- Affected URL or file
- Steps to reproduce
- Browser/device details if the issue is client-side
- Whether localStorage, service worker cache, or Cloudflare headers are involved

Do not include real secrets or personal data in a report. This app should not
collect either.
